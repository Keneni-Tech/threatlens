# ThreatLens

AI-powered security incident investigation built with Django and OpenAI

## Overview

ThreatLens helps security analysts transform raw security events into structured incident investigations using AI

The application analyzes security logs and provides:

- Incident summary
- Severity assessment
- Confidence score
- Evidence extraction
- MITRE ATT&CK mapping
- Investigation recommendations
- Containment actions

## Technology Stack

- Python 3.13
- Django 6
- OpenAI API
- Pydantic
- SQLite (development)
