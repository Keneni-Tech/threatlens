# Contributing

Thank you for your interest in ThreatLens.

This project is currently maintained by a single developer during OpenAI Build Week.

If community contributions are accepted in the future, the workflow will include:

1. Fork the repository.
2. Create a feature branch.
3. Write clear, well-tested code.
4. Run the test suite.
5. Submit a pull request with a concise description.

## Coding Guidelines

* Follow PEP 8.
* Prefer readability over cleverness.
* Keep functions focused and testable.
* Document public classes and methods.
* Never commit secrets or API keys.
