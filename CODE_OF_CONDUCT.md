# Code of Conduct

## Our Commitment

ThreatLens aims to foster a welcoming, respectful, and professional community.

This policy applies to repository issues, pull requests, reviews, and
other project spaces.

Participants are expected to:

* Be respectful.
* Be constructive.
* Be inclusive.
* Help others learn.

Unacceptable behavior includes:

* Harassment
* Personal attacks
* Hate speech
* Disruptive conduct

Project maintainers may remove content or restrict participation when necessary to maintain a positive environment.

## Reporting

Report conduct concerns privately to the project maintainer through a
private contact method listed on the
[Keneni-Tech GitHub profile](https://github.com/Keneni-Tech). Do not use
a public issue when doing so would expose the affected person or
sensitive details.
